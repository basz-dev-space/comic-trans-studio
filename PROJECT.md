# comic-trans-studio Project Documentation

## Purpose
comic-trans-studio is designed to streamline comic translation tasks in a single app, automating repetitive steps and providing AI-powered tools for translators and readers.

## Target Users
- Translators
- Readers seeking AI-powered translation

## Core Features
- Translation tools (AI-powered)
- Editing canvas for graphics (text, textbox, shape, etc.)
- OCR integration with mask creation for cleaning text
- Automated cleaning of old text from comic pages using OpenCV and LaMa
- Project and draft saving functionality

## Typical Workflow
1. Create a new project
2. Upload comic chapters as ZIP or PDF
3. Use translation tools and canvas editor to translate and edit pages
4. OCR extracts text and creates masks for cleaning
5. Automated cleaning removes old text from images
6. Save project or draft for later editing

## Automation
- Tasks like OCR, mask creation, and text cleaning are automated to reduce manual effort.

## Team Standards
- All agents must act as senior software developers, use skills for every task, plan step-by-step, and leave logs for collaboration.
